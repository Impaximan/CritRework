Update 1.6.0.0 "Locked and Loaded"
Additions:
-Added a new type of item, "Augmenations," which can be applied to weapons for altered critical strike behavior
-Added A.M.P.L, an augmentation which is dropped by Skeleton Commandos post-Plantera
-Added Epitaph, an augmentation dropped rarely by skeletons
-Added the Honeydipped Stinger an, augmentation which is crafted post-Queen Bee
-Added the Pocket Lightning Rod, an augmentation which is found in the Underground Desert
-Added the Boomstickifier, an augmentation sold by the demolotionist
-Added the Thorn Ring, an augmentation crafted with wood (and a bit of [HEART SHAPED OBJECT]...)
-Added the Buckler, an augmentation found in golden chests
-Added the Curser's Quill, an augmentation found in the Dungeon
-Added the Hunter's Drawstring, an augmentation which is dropped by Skeleton Archers
-Added the Manalytic Converter, an augmentation crafted with Copper/Tin
-Added the Blood Cog, an augmentation crafted using Crimtane
-Added the Imperfect Storm, an augmentation crafted using Demonite
-Added Apophis, who does not discriminate
-Added the Windwalker, an accessory found in Sky Chests
-Added Skyward Sling, a new accessory crafted with feathers and bronze
-Added the Contaminated Sapling, a new accessory crafted with jungle spores and glowing mushrooms
-Added a new weapon prefix, "Versatile," which allows you to put multiple augmentations on a weapon
-Added several new prefixes which can be given to augmentations, including prefixes which cause augmentations to only activate during certain conditions
-Added a new material, the "Greenprint," which drops from Lihzahrds in the Jungle Temple and can be used to create powerful, crit-powered constructs
-Added Construct Armor
-Added Reclaw potions
-Added Horrificracker
-Added Gilded Remains, a new utility item sold by the Pirate NPC
-Added Risky Token
-Added Homunculus Token

Changes
-Rewrote critical strike damage tooltips to say critical strike power instead (for consistency with augmentations and less misleading behavior)
-Starter Whetstones are no longer craftable (you only have the 2 that you started with now)
-'Vitriolic' prefix now inflicts you with On Fire! instead of Poisoned
-Boss HP multiplier is now 1.3x by default (can still be set down to 1)
-Reduced damage multiplier of holy whetstone
-Removed "Necromantic" prefix
-Added a tooltip for Bronze alloy
-Reforge hijack chance is now 20% at minimum
-Changed the value of Divine Ingots

Fixes
-Updated hitbox for overgrown Vilethorn and Crystal Vile Shard to grow as intended
-The Goblin Tinkerer will no longer make the hijack sound for items which cannot critically strike anyway
-Fixed Dungeon loot from this mod not spawning properly
-Fixed Prosthetic Arm damage bonus displaying incorrectly for the longest time
-Fixed weapons critically reforged with the Web Covered Whetstone sometimes sending you away faster than the speed of light, causing you to clip out of the world entirely
